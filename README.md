# 海洋風數獨 Ocean Sudoku

這是一個海洋風、手機可用的數獨網頁遊戲專案，使用 HTML、CSS、JavaScript 製作。

本版包含兩種分享玩法：

1. **分享同一題網址**：朋友打開後看到同一題，但進度各自獨立。
2. **多人即時房間**：需要設定 Firebase Realtime Database。建立房間後，朋友打開房間連結可以同步同一盤的填數字進度。

## 功能

- 自動產生完整合法數獨答案
- 挖空後檢查題目是否只有唯一解
- **20 段難度**：1 級最簡單，20 級最難
- **多主題風格**：海洋風、護眼風、電子紙、暮光風、櫻花風、森林風
- 手機版響應式介面
- 分享同一題網址：網址會帶有 `seed`、`difficulty`、`theme`
- 多人即時房間：房間代碼、房間連結、玩家列表、同步填數字進度
- 多人房間會分開統計每位玩家正確填入的格數
- **電腦解題教學**：從原題開始，一步一步說明候選數、唯一候選、隱性唯一與進階推理示範
- 筆記模式 / 候選數
- 提示功能
- 檢查目前盤面
- 錯誤、提示、時間統計
- 支援鍵盤操作
- 數獨盤面加入座標：上方 1～9 表示行，左側 a～i 表示列，例如 b4

## 檔案結構

```text
index.html                主頁面
styles.css                介面樣式
app.js                    遊戲邏輯、數獨產生器、多人同步、主題切換
firebase-config.js        Firebase 設定檔，需要貼上你的專案設定
firebase-rules-demo.json  Firebase Realtime Database 測試規則
README.md                 專案說明
```

## 單人使用方式

直接用瀏覽器打開 `index.html`，或部署到 GitHub Pages。

按下「分享同一題」後，遊戲會複製目前題目的連結，例如：

```text
index.html?seed=abc123&difficulty=12&theme=epaper
```

朋友打開這個連結會得到相同題目；這是「同題各自玩」，不會同步彼此進度。


## 電腦解題教學

按「開始教學」後，系統會從原題盤面開始示範解題。每按一次「下一步」，電腦會填入一格，並說明這一步的思考方式。

目前支援的教學說明包含。教學中的「同宮」代表同一個 3×3 小方塊，也就是包含目前格子的那一個九宮格區域。

- **唯一候選**：某格排除同行、同列、同一個 3×3 宮格後只剩一個候選數。
- **隱性唯一**：某一列、行或宮中，某個數字只剩一個位置可以放。
- **進階推理示範**：當基本技巧暫時推不動時，電腦會用候選數最少的格子示範較高階的推理方向。

為避免覆蓋大家的同步進度，多人房間中會暫停教學功能；請先離開房間再使用。教學進行時，教學面板會變成浮動視窗，可拖曳移動，方便對照盤面。

## 多人房間使用方式

多人房間需要先完成 Firebase 設定：

1. 到 Firebase Console 建立專案。
2. 新增 Web App。
3. 建立 Realtime Database。
4. 到 Realtime Database 的 Rules 貼上 `firebase-rules-demo.json` 的內容。
5. 複製 Firebase SDK config。
6. 打開 `firebase-config.js`，把 placeholder 換成你的實際設定。
7. 上傳更新後的檔案到 GitHub repository。
8. 等 GitHub Pages 重新部署。
9. 打開網站，按「建立房間」。
10. 按「複製房間連結」給朋友。

注意：`firebase-rules-demo.json` 是方便測試的公開房間規則，適合原型測試，不適合有帳號、排行榜、個資或正式商用資料的版本。

## 快捷鍵

- `1`～`9`：輸入數字或切換筆記
- `Backspace` / `Delete` / `0`：清除目前格子
- `N`：切換筆記模式
- `H`：使用提示
- `T`：開始電腦解題教學
- 方向鍵：移動選取格子

## 這一版更新

- 修正電腦版教學面板拖曳時的滑鼠偏移，並限制面板留在畫面可視範圍內，避免拖曳時跳走或消失。

- 難度從原本少數幾段，擴充成 **1～20 級**。
- 新增 **護眼風**、**電子紙**、**暮光風**、**櫻花風**、**森林風** 等主題。
- 新增 **電腦解題教學**，可逐步教使用者完成目前題目。
- 主題會記住上次選擇，並寫入分享連結。
- 手機版把「筆記模式、清除、提示、檢查」和 `1～9` 數字鍵盤移到多人房間區塊上方，操作更順手。
- 多人房間會分開顯示每位玩家正確填入的格數，例如「海友AN7L（你）｜3 格」。
- 只有手動填對的格子會計入玩家解題數；使用提示填入的格子不列入個人解題數。
- 如果正確格子被清除或改錯，該格原本歸屬玩家的格數會扣回。
- 修正「暮光風」主題的文字對比，讓標題、按鈕、輸入框、面板與數獨盤面更清楚易讀。


## 主題可讀性統一優化

這一版針對所有主題做了統一的可讀性檢查與調整：

- 所有主題的標題、說明文字、統計數字、按鈕、輸入框、下拉選單都提高對比。
- 數獨盤面的普通格、題目格、選取格、同列/同行/同宮提示色都改成更容易辨識的配色。
- 教學面板、多人房間、玩家標籤、座標軸也套用一致的高對比文字與背景。
- 暮光風保留深色氛圍，但盤面改成高可讀性的淺底深字，避免長時間解題時看不清楚。


## 本版拖曳問題修正

電腦教學面板不再使用自由拖曳，因為瀏覽器縮放、固定定位、手機安全區與頁面捲動同時作用時，容易造成游標與面板位置不同步。這版改成穩定的「位置切換」：開始教學後，可用「左上、右上、左下、右下」按鈕把教學面板停靠到不同位置，避免跳位或消失。

## 更新 GitHub Pages 時的提醒

如果你的 GitHub 上 `firebase-config.js` 已經填好 Firebase 設定，更新檔案時請保留它，或在上傳新版後再把 Firebase 設定貼回去。

## 模式切換更新

- 選擇「電腦教學」才顯示教學面板，並隱藏多人房間、筆記、提示、數字鍵盤等其他操作面板。
- 選擇「一般遊玩」會隱藏教學面板，並恢復其他功能畫面。
- 數獨盤面加入座標標示：上方為行 1～9，左側為列 a～i。
- 電腦教學中的說明改用座標，例如 `b4（第 b 列、第 4 行）`。
- 電腦教學面板在教學進行時可拖曳移動，避免遮住想觀察的盤面位置。


## 本版修正：電腦教學面板拖曳與縮放

- 電腦教學進行中，教學面板改回滑鼠拖曳移動。
- 使用面板上方「拖曳移動教學面板」區域移動，不會再混用右下角定位，減少跳位問題。
- 教學面板右下角新增縮放控制點，可拖曳調整面板大小。

## 本版修正
